# Boswindor 海外社媒封面生成器 — 产品需求文档 (PRD)

**版本**: V4.2.0  
**状态**: 已实现 (单文件应用 `Cover_V4.2.0.html`)  
**发布日期**: 2026-05  
**目标用户**: Boswindor 品牌运营团队 — 快速生成 IG Reels 竖屏封面图

---

## 1. 项目概述

### 1.1 产品定位
一款运行在浏览器端的 B2B 工业品牌社媒封面设计工具，单文件 HTML 应用，无需后端。面向 Boswindor (工业门/窗品牌) 的海外社媒运营需求，一键生成符合品牌规范的 1080×1920 IG Reels 封面图。

### 1.2 核心价值
- **零门槛**: 打开即用，无需安装任何软件
- **工业美学**: 内置 5 种工业 B2B 风格背板 (质检卡、3D描边、毛玻璃、建筑线框、微光投影)
- **所见即所得**: 右侧手机模拟器 1/3 比例实时预览，与导出 Canvas 完全一致
- **智能辅助**: AI 自动检测文字最佳位置 + IG 系统 UI 叠加层辅助避让
- **品牌规范**: 内置 Boswindor 品牌色板、字体、Logo 系统

### 1.3 技术栈
| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 1080×1920 封面渲染与导出 |
| Tailwind CSS (CDN) | 深色主题 UI 框架 |
| Google Fonts | Montserrat / Space Grotesk / Noto Sans SC |
| FontAwesome 6.5 | 图标系统 |
| Vanilla JavaScript | 全部交互逻辑，零依赖框架 |
| Playwright | E2E 测试 (88 个用例) |

## 2. 功能需求

### 2.1 三标签控制面板

#### 2.1.1 素材 (Media)
- **背景上传**: 支持点击上传和拖拽上传
  - 图片格式: JPG、PNG
  - 视频格式: MP4、MOV
- **Logo 上传**: 独立上传区域，支持图片拖拽
  - 自动尝试加载 `Boswindor 公司logo.png`
  - 加载失败显示未上传状态
- **视频截帧**: 视频背景模式下显示进度控制面板
  - 播放/暂停控制
  - 时间轴滑块选择定格画面
  - 实时时间显示 (mm:ss.ms 格式)
- **背景微调**:
  - 缩放: 100%-150%，步长 1%
  - Y轴平移: -150px ~ +150px
  - 实时影响预览和画布渲染

#### 2.1.2 文字 (Text)
- **内容编辑**:
  - 主标题: 英文文本输入，默认 "AS2047 Windows to Australia 25 Days Shipping"
  - 高亮词: 在标题中标记高亮的单词，默认 "Windows"
  - 副标题: 次要描述文本，默认 "Details Before Manufacturing"
  - 目标受众: 底部标签文本，默认 "For Custom Projects"
  - 页脚右侧文本
- **排版控制** (所有值在 Canvas 侧使用原始值，预览侧 1/3 缩放):

| 参数 | 控件类型 | 范围 | 默认值 (V4.0.0) |
|------|---------|------|------------------|
| 标题字号 | range | 48-96 | 88 |
| 标题字间距 | range | -8 ~ +4 | -2 |
| 标题行高 | range | 80-120 | 112 |
| 标题描边宽度 | range | 0-4 (step 0.5) | 3 |
| 副标题字号 | range | 20-56 | 46 |
| 副标题字间距 | range | -4 ~ +6 | 1 |
| 底部标签字号 | range | 18-48 | 34 |
| 底部标签字间距 | range | -4 ~ +6 | 1 |

- **字体选择**: Montserrat / Space Grotesk (默认)
- **高亮色选择**: 3 个预设色
  - 德系琥珀橙 #D47614
  - 博仕标准橙 #F3921F (默认)
  - 暖金色 #DE9F35
- **高亮词解析**: 大小写不敏感匹配，自动拆分主标题为 前段-高亮-后段 三部分
- **边界处理**: 空高亮词不崩溃、高亮词不在标题中不崩溃、全空标题不崩溃

#### 2.1.3 样式 (Style)
- **文字背板样式** (5 种):

| 样式 | 效果 |
|------|------|
| 质检卡 (inspection) | 左侧橙色竖线 + 微描边 + 扩散阴影，匹配工业品检报告风格 |
| 3D描边 (stroke) | 粗黑描边 + 四方向阴影，厚重立体感 |
| 毛玻璃 (glass) | backdrop-blur + 半透明圆角卡片 + 白色微边框 |
| 建筑线框 (architect) | 左侧橙色 4px 边框 + 轻微毛玻璃 + 圆角 |
| 微光投影 (shadow) | 仅保留文字投影 |

- **压暗保护** (inspection 模式专用): 4 级 — none / light / medium / heavy
  - 影响顶部和底部渐变遮罩的透明度
- **背景透明度**: 0%-100%，影响背板背景不透明度
- **AI 智能定位**:
  - 自动分析背景画面复杂度 (Canvas 像素采样)
  - 默认锁定 1300px 黄金下三分之一区
  - 手动拖拽后 AI 徽章自动隐藏
  - 支持手动触发重分析

### 2.2 手机预览区

#### 2.2.1 预览渲染
- **尺寸**: 模拟 IG Reels 竖屏手机，按 Canvas (1080×1920) 的 1/3 比例缩放
- **文字定位**: Y 位置使用百分比映射 (`(titleY - 180) / 1920 × 100%`)
- **背景渲染**: 图片/视频 aspect-fill 裁剪，支持缩放和 Y 轴平移变换

#### 2.2.2 拖拽交互
- 在预览区域内垂直拖拽调整文字 Y 位置
- 拖拽时 Y 滑块实时联动
- 拖拽结束触发 Toast 提示
- 手动拖拽后 AI 徽章隐藏

#### 2.2.3 IG 系统叠加层
- **默认可见**: 模拟 Instagram Reels UI 元素
  - 用户名 "boswindor_doors"
  - Reels 标识
  - 互动数据: 点赞 4.8万、评论 1,368
  - 分享按钮
  - FOLLOW 按钮
  - 音乐标签 "Original Audio"
- **切换按钮**: 点击切换显隐 (opacity 0/1 过渡)
- **目的**: 辅助用户避开 IG 原生 UI 遮挡区域

#### 2.2.4 安全线
- **默认隐藏**: 1:1 方形虚线框 (橙色边框 + 半透明填充)
- **切换按钮**: 点击切换显隐
- **设计目的**: 标识 IG Feed Grid 1:1 裁剪区域

### 2.3 B2B 预设模板

4 个工业场景预设，一键切换主标题、高亮词、副标题、受众文字和背景图片:

| 预设 | 场景 | 默认背景 |
|------|------|---------|
| shipping | 集装箱海运实景 | Unsplash 物流仓库 |
| inspect | 产线品检特写 | Unsplash 品质检测 |
| showroom-pull | 展厅透视全景 | Unsplash 建筑展示 |
| meeting-neck | 设计院会审 | Unsplash 会议室 |

- 每个预设包含 Unsplash 高清图片 + 备用 URL (FALLBACK_URLS)
- 切换模板时自动将 Y 位置复位至 1180px，然后触发 AI 重分析
- 启动时自动加载 shipping 预设

### 2.4 品牌系统

#### 2.4.1 品牌色板
| 颜色 | 色值 | 用途 |
|------|------|------|
| 品牌橙 | #F3921F | 主强调色、Logo、高亮元素 |
| 暖黄 | #FFC800 | 辅助强调 |
| 纯白 | #FFFFFF | 文字主色 |
| 深黑 | #0A0A0A | 页面背景 |
| 灰黑 | #121212 | 卡片/面板背景 |
| 边框灰 | #222222 | 分割线/边框 |

#### 2.4.2 品牌字体
| 字体 | 用途 |
|------|------|
| Space Grotesk | 主标题 (800 weight) / 副标题 (700) |
| Montserrat | 底部标签 (700) |
| Noto Sans SC | 中文 UI 标签 |

#### 2.4.3 Logo 系统
- **预览区**: logo 图片 + 白色竖向分隔线 + "BOSWINDOR" 文字
- **开关控制**: 勾选后在预览区显示 Logo
- **上传支持**: 独立 Logo 上传区域，默认尝试加载本地文件

### 2.5 Canvas 导出

#### 2.5.1 渲染规格
| 参数 | 值 |
|------|-----|
| 画布尺寸 | 1080 × 1920 px |
| 输出格式 | JPEG |
| 输出质量 | 0.95 |
| 文件名 | `Boswindor_Cover_Template_{presetId}.jpg` |

#### 2.5.2 渲染流程
1. **背景绘制**: aspect-fill 缩放裁剪 + 用户缩放/平移变换
2. **渐变遮罩渲染** (非 vignette 模式):
   - 顶部 240px 渐变 (黑色 15% → 透明)
   - 底部渐变 (透明 → 黑色 70% → 黑色 98%)
3. **文字 V4.0.0 效果** (全部背板样式):
   - 副标题与底部标签之间绘制分隔线 (白色 12% 透明度, 150px 宽)
   - 底部标签包裹在圆角 pill 背景中 (rgba(8,8,8,0.85) + 白色 10% 边框)
4. **文字排版**: 最大宽度 720px，自动折行，支持 letterSpacing
5. **描边/阴影**: 根据背板样式应用不同的描边宽度、阴影模糊和偏移

#### 2.5.3 V4.0.0 画布增强
- Canvas letterSpacing 参与词宽测量 (`ctx.letterSpacing`)
- 非 inspection 样式应用标准顶部/底部阴影渐变
- 全部背板样式统一添加分隔线 + badge pill 背景
- 文字颜色统一提升为 #FFFFFF (原 #CCCCCC)

### 2.6 Toast 通知系统
- **类型**: success (橙色) / error (红色)
- **行为**: 淡入 → 3 秒显示 → 淡出移除
- **触发场景**:
  - 启动加载默认预设完成
  - 图片/视频上传成功/失败
  - Logo 上传成功
  - 导出成功
  - AI 分析完成
  - 模板切换
  - IG 叠加层切换
  - 手动拖拽完成

### 2.7 交互细节
- **Header**: 固定顶部，显示品牌 BW Logo、标题 "BOSWINDOR GROWTH ENGINE"、绿色状态指示灯
- **三标签切换**: 素材/文字/样式，高亮状态橙色背景
- **滚动条**: 自定义暗色风格，悬停变为橙色
- **手机预览框**: 圆角容器 + 内阴影模拟手机质感
- **导出按钮**: 橙色醒目按钮 "生成并下载"

## 3. 非功能需求

### 3.1 浏览器兼容性
- 测试目标: Microsoft Edge (Playwright 配置)
- 依赖 CDN (Tailwind CSS, Google Fonts, FontAwesome)，需网络连接

### 3.2 性能
- 单文件 HTML 约 123KB
- Canvas 渲染在 `document.fonts.ready` 后执行，确保字体加载完成
- 图片预加载检测 (`complete` 状态)

### 3.3 错误处理
- DOM 元素 null-guard 全覆盖 (T07 标注 6 个已知缺失元素，均有 fallback)
- 视频格式不支持 → Toast 错误提示
- Logo 加载失败 → 显示 "未上传" 状态 + 备选占位
- CORS 限制图像分析 → 静默 fallback 至默认位置
- 空标题/空高亮词/不匹配高亮词 → 页面不崩溃

### 3.4 测试覆盖
- **88 个 Playwright 测试用例** (见 `tests/cover.spec.js`)
- 覆盖: 静态结构、预设行为、文本控制、背景控制、背板样式、IG叠加层、Logo、文件上传、视频控制、导出、错误边界、拖拽交互、B2B模板、副标题排版、预览画布一致性
